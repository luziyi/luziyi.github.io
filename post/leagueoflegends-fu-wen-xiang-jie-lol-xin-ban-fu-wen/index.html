LeagueOfLegends符文详解(LOL新版符文)

· 2023-01-19 · # LOL

lol符文就是微量增强和调整你属性的东西。根据你常用的英雄，来购买你所需要的。说通俗点就是攻击力，法术强度，固定护甲，固定魔抗，成长攻击力，成长护甲，成长魔抗，攻击速度等等的这些属性。游戏中默认可以装备2套不同属性的符文，也称符文页，可以随意的切换2个符文页。

1. lol新版符文介绍

五个符文，分别为精密、主宰、巫术、坚决、启迪。
玩家可以为英雄选择一个主符文系列和一个副符文系列，前者可以选择1个基石天赋、1个高级符文和2个符文，后者可以选择2个符文或高级符文。
精密和主宰各有4个基石天赋，巫术、坚决和启迪各有3个基石天赋，共有17个基石天赋。
需要注意的是，每个符文系列的符文会被分为3行，每一行中的符文是不能多选的

2. LOL新版符文

种类：符印、雕文、印记、精化。功能：攻速、护甲穿透、攻击力、生命值、魔抗、护甲、法强、法术穿透。其中精华有3个槽。其它的都有9个槽。精化与印记中的状态加攻击比较多最好选择攻击，雕文与符印中的状态加防御比较多最好选择防御。
具体符文的选择搭配可以按使用英雄的类型来分：
坦克类英雄——主要以护甲、魔法抗性、生命值等属性符文搭配为主。
DPS类英雄（物理）——主要以攻击力、攻击速度、暴击率、暴击伤害、护甲穿透等属性符文为主。
DPS类英雄（法术）——主要以降低技能冷却时间、法力回复、法术强度、法术穿透等属性符文为主。

3.基石符文及普通符文介绍

精密系

首先是精密系的符文，精密系的特性是强化攻击和持续伤害。比较适合ADC或者AD刺客这类的英雄。

基石符文

强攻：用3次连续的普攻命中一名敌方英雄时，将使其变为易损状态，造成额外伤害并使所受的来自任意来源的伤害变多，持续6秒。

类似于战争热诚，无限叠加的的攻击可在持续的战斗中提供非常可观的输出。非常适合ADC的一个基石符文。

致命节奏：在伤害一名英雄1.5秒后，你会获得大幅攻击速度加成。甚至你可以暂时溢出你的攻速上限，即突破2.5攻速。

通过攻击敌人叠加攻速，在短时间内突破攻速上限。对于大嘴剑圣等短时间内爆发极高并且较为以来攻速的英雄有着极大的提升。

迅捷步法：攻击和移动会积攒能量层数。在100层时，你的下次攻击会治疗你并提升移动速度。

对线型技能，适用于对线不强的英雄，换血时利用加速可让多消耗对面，叠满层数后可回血增加英雄的赖线能力。团战时的作用要小于另外两个基石。

普通符文

过量治疗：你身上的溢出治疗效果将变成一个护盾。

解析：吸蓝刀饮血剑效果融入其中，让AD的续航能力大大加强。溢出的治疗量会成为AD的护盾，当AD的蓝量低于一定程度时不断攻击会回复AD自身的蓝量。蓝药取消之后，部分AD选择做出吸蓝刀来保证蓝量的回复。如今有了符文之后，AD的蓝量回复问题得到了解决，可以更加轻松在线上使用技能压制对方。

凯旋：参与击杀会回复你15%的已损失生命值并提供额外25金币。

死亡游戏天赋改版，参与击杀的英雄可以获得了额外的赏金。作为辅助英雄，比较缺钱。改版之后的类似死亡游戏的天赋，只要辅助参与了击杀获得了助攻，也能获得金钱的奖励。使辅助英雄的经济得补贴。

气定神闲：在升级或参与击杀后的5秒里，你的法力花费将得到完全返还。

类似于鬼书的被动，在回蓝方面有着不错的效果。技能型AD比较依赖蓝量，可以选择这个符文。

传说：欢欣：参与击杀敌人将提供永久的攻击速度。

攻速可以永久叠加，AD的后期能力大大增强。尤其对于女警这种比较依赖攻速的ADC来说更是必备的。

传说：韧性：参与击杀敌人将提供永久的韧性。

永久叠加韧性。非常适合一下AD战士和刺客。

传说：血统：参与击杀敌人将提供永久的生命偷取。

永久叠加的吸血无论是对刺客还是ADC而言都是不错的选择。

致命一击：对低生命值的敌方英雄造成更多伤害。

这个天赋就是刺客用来增加输出的，比较适合高爆发英雄。

砍倒：对最大生命值多于你的所有敌方英雄造成更多伤害。

符文中自带小轻语效果，AD杀坦能力稍微增强。新版基石天赋中不仅增强了AD的反杀能力，同时也让AD的打坦克能力稍微获得了增强。对方的生命值和你的差额可以让AD多打出8的额外伤害，让AD在轻语做出前打前排的能力稍微增强。

坚毅不倒：当你的生命值较低时，你的攻击对英雄多造成伤害。

生命值低于80％最多增加5％的伤害，这个天赋是触发几率最大的天赋，如果是血量越低伤害越高的话，那么这个天赋只适合蛮王剑魔这类走钢丝英雄。

主宰系

基石符文

电刑：在3秒内用3个独立的攻击或技能命中一位英雄时，会造成额外的自适应伤害。

新改版的雷霆，增加爆发伤害。适合高爆发英雄。

掠食者：为你的鞋子添加一个主动效果，此效果可提供一个巨量移动速度加成，并使你的下次攻击或技能造成额外的自适应伤害。

对于移速有较大需求的英雄的首选，如人马稻草人等英雄。

黑暗收割：英雄、大型小兵、大型野怪会在死亡时掉落灵魂精华。触碰一个灵魂可拾取它，并使你的下次攻击造成基于已收集灵魂精华数的额外自适应伤害。

比较适合喜欢刷的英雄使用，累计灵魂越多额外伤害越高，适合剑圣死歌等英雄。

普通符文

恶意中伤：对移动或行动受损的敌人造成额外真实伤害。

比较合适ADC，有红buff时，配合起来增加不错的真实伤害。

血之滋味：在你伤害一名敌方英雄时治疗自身。

点这个技能虽然牺牲输出，但是让自身有了不错的续航的能力，比较适合高爆发但是缺少续航能力的英雄。

猛然冲击：在使用一次突进、跃击、闪烁、传送效果或离开潜行状态后获得穿甲和法术穿透。

闪现和传送CD很长，所以适合有突进或者跃击的英雄，那么就是适合武器、瑞雯、杰斯、鳄鱼这样的战士天赋了。

僵尸守卫：在击杀一个敌方守卫后，该守卫的位置上会升起一个友方的僵尸守卫。当该守卫的持续时间耗尽后，其仍然会复活为一个僵尸守卫。

在排眼之后会留下友军的视野，视野的压力相对得到了缓解。新版基石僵尸守卫天赋，当你排掉对方的侦查首位时，在原地则会留下一个暂时的僵尸守卫，能持续180秒同时并不算作你自己的侦查首位数量。当你排除掉对方的视野时，等于变相为自己增加了一个视野。这样一来就可以大大缓解了己方的视野压力。

幽灵魄罗：当你进入草丛时，一个魄罗会出现。它会呆在后面来为你提供视野。

这个技能有点类似远见改造，没被敌人发现就一直存在的视野，在比较刁钻的地方，是有可以存在一整局的。

眼球收集器：在参与击杀英雄和守卫时收集眼球。每个眼球都会提供永久的攻击力或法术强度，自动适应，并在完成收集后会提供额外加成。

这是一种加成属性，类似于小法Q叠加法强一样。比较适合螳螂这种打野刺客反野清视野不管是杀人还是清野都能有不错的攻击力加成。

贪婪的猎手：独特的参与击杀会永久使来自技能的伤害提供治疗效果。

增加自身续航能力，杀人提升治疗效果让高伤害法师获得更好的续航能力，不至于被耗一下就要回家。

灵性猎手：独特的参与击杀会提供永久的主动道具冷却缩减(包含饰品效果)。

杀人获得装备减CD，适用于出幽梦这类刺客英雄和妖姬这种出科技枪的法师刺客。

无情猎手：独特的参与击杀会提供永久的非战斗状态移动速度加成。

比较适合短腿的英雄，让自己更好的进行跑图支援。

巫术系

基石符文

召唤：艾黎：你的攻击和技能会向目标派出艾黎，来对目标敌人造成伤害或为友军套上护盾。

一个全新的概念符文，非常棒。既能对敌人造成额外伤害，又可以为友军施加保护。所以很明显，这是功能性法师的首选符文。比如拉克丝、卡尔玛、璐璐等。

奥术彗星：用一个技能对一个英雄造成伤害时，会在其位置处召唤一颗彗星，若【奥术彗星】尚未冷却时，则改为减少它的剩余冷却时间。

对于远程消耗型的英雄很好的提高了英雄的输出和爆发能力。如泽拉斯维克托等英雄，因彗星有延迟，配合冰杖效果更佳。

相位猛冲：在2秒内用3个独立的攻击或技能命中一位敌方英雄时，会提供爆发性的移动速度。

风暴骑士的加强版，触发条件简单，不管是追杀还是逃跑都是不错的选择。

普通符文

无效化之法球：当你在承受即将使生命值降至危险线的魔法伤害时，提供一个魔法伤害护盾。

当你在承受即将使生命值降至危险线的魔法伤害时，提供一个魔法伤害护盾。

法力流系带：永久性地使你使用的下一个技能返还它的法力或能量消耗并为你回复一些已损失的法力或能量。

增加续航类符文。完美CD情况下，每60秒大约能为你提供100左右的法力回复。这相当于2层腐败药水回复的法力值。中单刷子的最爱比如说发条泽拉斯维克托。

终极技能帽：你的终极技能的冷却时间得到了缩减。并且你每次使用终极技能时，它的冷却时间就会得到进一步缩减。

大招流AP首选，例如安妮的熊团控，最高可以获得15％-CD。任何吃大招的英雄，都可以点这个。

超然: 在你到达10级时提供10%冷却缩减。溢出的冷却缩减属性会转化为攻击力或法术强度，自动适应。

冷却缩减不怕溢出，能转化为法强。过去冷却缩减最多只有40，多出的冷却缩减未能发挥他们的作用。但是，新版的基石天赋下，冷却缩减溢出之后，可以转化为法强或者是攻击力。这样一来法师的冷却缩减再也不怕溢出了。

迅捷：提供4%额外的移动速度。基于你的额外移动速度提供额外的法术强度或攻击力自动适应。

适合短腿法师，或者说吃移速的英雄例如卡牌瑞兹这类需要移速的英雄，来更好的跑图支援。

绝对专注：在高于70%生命值时，获得额外的自适应伤害。

在生命保持一个安全状态时，提供一个不错的属性，比较百搭。

焦灼：每过20秒，你的第一个技能在命中时会使敌方英雄燃烧。

对线期符文，类似于冥火，但是有20——8秒CD，单体性价比很一般。搭配其他消耗性符文可以使用，适用于有指向性技能的英雄，对线期的压制。

水上行走：在河道时提供移动速度和法术强度或攻击力，自动适应。

这组符文可以让你在适当的情况下获得伤害提升。你的选择有：使自己的首个技能灼烧敌方英雄，在河道中获得额外攻击属性，以及获得随时间呈指数增长的攻击属性。

风暴聚集：随着游戏的进程不断提供攻击力或法术强度，自动适应。

人人自带N条火龙属性?新版的基石风暴聚集能随着时间的不断叠加而不断叠加你的法强或者是攻击力，当游戏进入到50分钟时，你就会自带120点法强，自带一顶帽子。当然前提是游戏能进入到那么后期的话…….

坚决系

基石符文

不灭之握：每过4秒，你对一个英雄发起的下一次攻击将造成额外魔法伤害、治疗效果、并永久提升你的生命值。

这个符文非常适合线上换血掺杂普攻的英雄。如今主流的带不灭之握的坦克英雄依然可以沿用新版不灭之握。另外一些普攻换血平凡的输出型英雄也可以点这个符文来增加后期的生存能力。

余震：在你定身一个敌方英雄后提供防御属性，稍后会在你的周围造成爆发性的自适应伤害。

加强版巨像，不仅能提供护盾还有爆发伤害。在硬控命中对方之后，能增加双抗，同时还能基于自己的生命值打出爆发伤害。坦克的低伤害时代一去不复返。

守护者：守护你选作技能目标的友军和那些距离你非常近的友军们。如果你和一个被守护的友军即将承受伤害，那么你们两个都会获得加速效果和一层护盾。

辅助的首选，增加了对线的容错率，在换血上也可有一些优势。保护型英雄都可以选择此天赋。

普通符文

坚定：你每有一个尚未冷却完毕的召唤师技能，你的韧性和减速抵抗就会获得提升。在施放一个召唤师技能后，立刻获得幅度更大的韧性和减速抵抗，持续一段不短的时间。

非常适合闪现开团的英雄。像皇子的EQ闪这样的开团再也不怕进去被秒杀了。

爆破：当位于防御塔附近时，持续充能一次针对防御塔的强力攻击。

坦克的拆塔能力增强。坦克上单在有段时间内不被待见就是因为拆塔速度太慢，导致单带难以牵制对方。而新版的符文中，基于坦克的生命值，能发动一次对防御塔的冲击，生命值越高，伤害越高，坦克的拆塔能力大大增强。

生命源泉：对一名敌方英雄施加移动受损效果会将其标记。你和友方英雄在攻击被标记的英雄时会获得治疗效果。

这个符文首先是适合打野，特别是拿着红BUFF的打野，其次是适合带有减速的英雄，如果下路带应该是一人带就够了，无需多带。

钢铁皮肤：获得+5护甲。治疗（含消耗品）会使你的护甲暂时提升5%。

3秒时间确实是很短，5的提升也不高，但是在下路如果有奶妈、带圣物之盾的辅助、琴女这些英雄的话，带这个对拼还是有点好处的。

镜之壳：获得+5魔法抗性。治疗（含消耗品）会使你的魔法抗性暂时提升5%。

同样的道理但是我觉得这个符文会上冷板凳，因为下路对拼魔抗需求不大，另外3点魔抗加成不多、5也不多，3秒时间还很短。

调节：在10分钟时获得+8护甲和+8魔抗，并使你的护甲和魔抗提升5%。

抗性成长性符文。相对上2的优点是不需要触发条件，永久生效。缺点是游戏前10分钟零作用。如果对线压力不大的坦克，可以选择。

过度生长：你附近每有小兵或野怪死亡时，提供永久最大生命值加成。

生命值无限叠加。当有大量小兵或者是野怪在附近被击杀时，坦克英雄会不断吸收，不断叠加自己的生命值。这样坦克英雄在带线时的收益会大大增加，人人皆有可能成为大虫子。

复苏：治疗效果和护盾效果会增强5%，并且如果目标的生命值较低，则此数值会提升额外的10%。

护盾治疗型辅助首选。有点类似之前辅助天赋风语者的祝福，只有辅助英雄才会点这个。

复苏之风：在受到来自一名敌方英雄的伤害后，会持续治疗你一些已损失生命值。

比较适用于没有续航能力的肉上单血越多续航能力越强。

启迪系

基石符文

启封的秘籍：获得“召唤师碎片”并可在单局游戏期间用它们在商店中更改你的召唤师技能。你的召唤师技能会拥有更短冷却时间。

召唤师技能可以更改，打龙必备?新版天赋出现了一点非常有趣的，就是召唤师技能可以随意更换。但是不能换成两个一模一样的技能。或者，和朋友5人一起开黑时，在大后期决战可以出现5个惩戒一起打大龙?

冰川增幅：你对敌方英雄发起的第一次攻击会使该英雄减速(一段时间内无法重复作用于相同目标)。用主动型装备使敌方英雄减速，还会对着他们发射冻结射线，并创造出一个缓慢消失的减速场。

技能自带冰杖效果，浪法维克托等技能衔接快的英雄的福音，现在冰杖减速效果减弱，配上这个天赋的话效果会好很多。
