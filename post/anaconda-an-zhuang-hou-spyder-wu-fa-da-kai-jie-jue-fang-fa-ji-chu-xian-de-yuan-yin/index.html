Anaconda安装后spyder无法打开解决方法及出现的原因

· 2023-02-04 · # Tech

无法打开spyder的可能原因：1.更新包之后，使得pyzmq的版本过高 2.就是安装时pyzmp的版本就过高了，导致无法打开spyder

解决方法：

打开Anaconda Powershell Prompt，输入命令spyder --new-instance查看报错

出现报错的情况：

第一种错误

Warning:
QT_DEVICE_PIXEL_RATIO is deprecated. Instead use:
QT_AUTO_SCREEN_SCALE_FACTOR to enable platform plugin controlled per-screen factors.
QT_SCREEN_SCALE_FACTORS to set per-screen factors.
QT_SCALE_FACTOR to set the application global scale factor.”

出现这种情况，说明你可能有一个QT的环境变量可以移除
所以解决的方法是用值auto来覆盖原有值

第二种错误：

AttributeError: module ‘xxx’ has no attribute ‘xxx’

根据大佬经验贴，在Anaconda Prompt中依次输入以下内容：

conda remove spyder-kernels
conda install spyder-kernels=0.*
conda install spyder=3.3.1

第三种错误

Bad file descriptor (C:\········)

遇到这种报错时上网查资料是说是pyzmq版本过高
第一步：进入Anaconda Powershell Prompt，输入命令pip uninstall pyzmq ，卸载高的版本。
注意：如果出现python PermissionError: [WinError 5]拒绝访问的情况，说明你的python权限不够
那么就按照下面的步骤操作：找到安装python的位置（默认是C:\ProgramData\Anaconda3），找到python应用程序


第二步：卸载完成后在Anaconda3 Powrshell中输入pip install pyzmq==19.0.2 安装老的版本后spyder就能运行了
